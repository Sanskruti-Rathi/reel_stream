# 🎬 ReelStream – Short Video Sharing App

ReelStream is a modern short video sharing mobile application built using Flutter and Firebase.  
It allows users to upload, view, and interact with short videos, similar to TikTok and Instagram Reels.

---

## 🚀 Features

- 🔐 User Authentication (Email & Password using Firebase)
- 🎥 Upload short videos with caption
- 🗜️ Video compression and thumbnail generation
- ❤️ Like videos and comments
- 💬 Comment on videos
- 👍 Like comments
- 👥 Follow / Unfollow users
- 🔍 Search users
- 👤 User profile with statistics (followers, following, posts, likes)
- ☁️ Video storage and delivery using Cloudinary
- ⚡ Smooth scrolling video feed (optimized playback)
- 📱 Clean and responsive UI
